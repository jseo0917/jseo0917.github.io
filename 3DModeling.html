3D Modeling

In the 3D Modeling, We can use OpenSCAD and Sketch Up to design rockets and houses. OpenSCAD is specialized software tailored for crafting robust 3D CAD objects.

Creating a house using OpenSCAD

Through a series of intuitive commands, you can both construct and preview intricate 3D models, as exemplified in the image on the left and right.

Creating a rocket using OpenSCAD

Furthermore, OpenSCAD empowers you to construct intricate objects of greater complexity, such as the rocket displayed on the left. In this example, we skillfully amalgamated various shapes, colors, and sizes to fashion the rocket, showcasing the software's versatility and creative potential.

Creating a house using Sketch Up

In contrast to OpenSCAD, SketchUp is a user-friendly software that allows you to create 3D objects without the need for manual command inputs. Additionally, it offers the convenience of accessing pre-built 3D objects created by other users, expanding your design possibilities. The pictures below are the example.

Creating a rocket using Sketch Up

Here is the rocket model that was created using SketchUp. Once you've designed it in SketchUp, you also have the option to 3D print it, as depicted in the image on the right.
